e:T861,<h1>Welcome to Yozma - A Simple and Automated Blog Platform</h1>
<p>Yozma is a minimalist blogging platform designed to be fast, automated, and easy to use. It leverages <strong>Next.js (Static Export Mode)</strong>, <strong>Markdown</strong>, and <strong>GitHub Pages</strong> to provide a seamless blogging experience without the need for complex backend management.</p>
<h2>How Yozma Works</h2>
<ol>
<li><strong>Write in Markdown</strong>: Each blog post is a simple <code>.md</code> file stored in the <code>posts</code> folder.</li>
<li><strong>Automatic Deployment</strong>: When a new Markdown file is pushed to the repository, <strong>GitHub Actions</strong> triggers an automated deployment.</li>
<li><strong>Static Site Generation</strong>: Using <strong>Next.js 15 (App Router)</strong> in <strong>Static Export Mode</strong>, the blog is pre-rendered for optimal performance.</li>
<li><strong>Hosting with GitHub Pages</strong>: The statically generated site is served for free using <strong>GitHub Pages</strong>.</li>
<li><strong>Beautiful Typography</strong>: Styled with <strong>Tailwind CSS Typography</strong>, ensuring a clean and readable design.</li>
<li><strong>Telegram Instant View</strong>: Supports <strong>Telegram Instant View</strong>, making posts instantly readable within Telegram.</li>
</ol>
<h2>Why Yozma?</h2>
<ul>
<li><strong>No database</strong> – Purely static content means no backend maintenance.</li>
<li><strong>Version-controlled</strong> – Blog content is managed directly via <strong>GitHub</strong>, ensuring easy collaboration and history tracking.</li>
<li><strong>Fast and SEO-friendly</strong> – Static export provides excellent speed and performance.</li>
<li><strong>Automated workflow</strong> – No manual deployment; push your post, and it's live!</li>
</ul>
<p>Yozma is perfect for developers and writers who want a simple, efficient, and automated blogging experience. Stay tuned for updates and improvements!</p>
<hr>
<p><strong>Get Started:</strong> Simply add a Markdown file to the <code>posts</code> folder and push it to GitHub—your blog will be updated automatically!</p>
